1. 国内生产总值（GDP）

国内生产总值（Gross Domestic Product，GDP） 是衡量一国经济规模最常用的指标。

完整定义：在一个国家地理边界内，在给定时期内，所有最终商品和服务的市场价值。

定义中每个词都有特定含义，缺一不可：

  • 市场价值（Market Value）：用价格来衡量产出，而不是数量。同样生产一个苹果和一辆汽车，对GDP的贡献完全不同。市场价格反映了人们对该商品的主观估值，是衡量实际价值的最佳代理指标。
  • 最终商品和服务（Final Goods and Services）：只计算供最终使用的商品，不计算生产过程中的中间品（Intermediate Goods）。棉花→棉线→棉布→牛仔裤这条生产链中，只计算最终的牛仔裤价值，而不是将各环节的价值重复累加——否则会造成重复计算（Double Counting）。
  • 国境内生产（Produced Within a Country）：只要在该国境内生产，不论生产者是哪国人或哪国企业，都计入该国GDP。这与 GNP（国民生产总值） 不同——GNP按国籍计算，GDP按地理位置计算。
  • 给定时期（In a Given Period）：通常以年度或季度为单位。同一商品不会在不同时期被重复计入。

不计入GDP的情况：

不计入的类型原因例子
中间品避免重复计算制造汽车用的钢铁
旧商品转手生产发生在过去，已计入当时GDP二手房交易、二手车
转移支付没有新的商品或服务被生产社会保障金、退休金、失业救济
家庭内部生产没有市场交易自己做饭、照顾子女
非法活动无法追踪地下经济

⚠️ 转移支付（Transfer Payment） 是常见混淆点：政府给退休人员发养老金，不计入GDP；政府雇人修路，计入GDP。区别在于是否有新的商品或服务被生产。

1.1 循环流量模型（Circular Flow Diagram）

循环流量模型建立在两个简化假设上：

  1. 家庭拥有所有生产要素
  2. 企业生产所有商品和服务

现实中这两个假设都不完全成立——大量生产要素由企业持有，家庭也会自行生产部分商品和服务（但不进入市场）。这个模型的价值在于揭示逻辑结构，而非精确描述现实。

在此模型中，资金与实物流向相反，形成闭环：

  • 实物流（顺时针）：生产要素从家庭流向企业；商品和服务从企业流回家庭。
  • 货币流（逆时针）：消费支出从家庭流向企业；要素收入（工资、租金、利息、利润）从企业流回家庭。

在这个封闭循环里，因为家庭是所有生产要素的唯一提供者，也是所有商品服务的唯一购买者：

  • 家庭支出 = 企业收入
  • 企业收入 = 企业成本（工资+租金+利息）+ 利润 = 家庭收入

⚠️ 第二个等式成立的前提是"利润也归家庭所有"——即企业由家庭拥有，利润最终流回家庭。这是模型假设，不是普遍事实。

因此在此模型中：总支出 = 总收入 = GDP，以下三种测量GDP的方法结果相同。

1.2 三种测量GDP的方法

三种方法从不同角度观察同一个经济活动，结果完全等价。需求侧（C+I+G+X-M）和生产侧（耐用品/非耐用品/服务/建筑/存货）加总结果也完全相同——因为每笔市场交易必须同时有买方和卖方，购买的总额必然等于生产的总额。

方法一：支出法（Expenditures Approach）

最常用的方法。把经济中所有购买最终商品和服务的支出加总：

Y = C + I + G + (X - M)

各组成部分详见1.3。

方法二：收入法（Income Approach）

把所有生产要素的收入加总：

Y = w + i + r + p
符号含义
w（wages）劳动获得的工资
i（interest）资本获得的利息
r（rent）土地获得的租金
p（profit）企业家才能获得的利润

方法三：增值法（Value-Added Approach）

把生产链各环节的增加值加总。

以牛仔裤为例：

环节产出市场价值增加值
棉农$10$10
棉线厂$20$10
布料厂$30$10
牛仔裤厂$50$20
合计$50

增值法的优势：适用于跨国生产链，只计算在本国境内新增的价值，避免重复计算其他国家已创造的价值。

例： 一件在中国生产零部件（价值$30）、在美国组装成品（最终售价$50）的商品。增值法下，中国GDP增加$30，美国GDP增加$20（只计算在美国境内新增的价值）。若用最终品法，只有美国的$50计入美国GDP，中国的$30不重复计算——两种方法结果一致，但增值法在追踪跨国供应链时更清晰。

1.3 支出法的组成部分

C：消费（Consumption）

家庭对最终商品和服务的支出，不含新房购买。

消费是GDP中最大的组成部分，在美国约占68%，且相对稳定，不会大幅波动。

包括：食品、服装、汽车（私人用途）、医疗、教育、娱乐等。

⚠️ 教育支出在经济学GDP核算中属于C（消费），而非投资——尽管日常语言中我们视教育为"投资"。区分经济学定义和日常语言定义很重要。

I：投资（Investment）

企业对资本品的支出，加上家庭新房购买。

I = 企业资本支出（设备、软件、厂房）
  + 企业存货变动（Inventory）
  + 家庭新房购买

⚠️ 经济学意义上的"投资"≠日常语言中的"投资"：

日常语言经济学GDP核算
买股票投资不计入GDP（只是资产转手）
买债券投资不计入GDP
企业买设备投资计入I
企业增加库存—计入I
家庭买新房—计入I
家庭买二手房—不计入GDP

投资是GDP中波动最大的部分，在美国约占17%。新技术或新产品可以推动投资激增，但信心下滑时投资可能骤然收缩——这正是GDP出现周期性波动的重要原因之一。

存货（Inventory）的跨期处理：

若生产链在计算期内未完成（如棉线在年底仍在仓库中），该中间品的市值计入当期GDP（视为投资）；下一期完成最终品后，只计算新增价值（最终品价值减去已计入部分），避免重复计算。

例： 棉线（$20）在第一期末仍在库存中，第一期GDP增加$20。第二期棉线被加工成牛仔裤（$50），第二期GDP增加$30（= $50 - $20），而非$50，否则$20会被重复计算。

为什么家庭购买新房计入I而非C：

新房是耐用资本品，建成后会在未来数十年持续提供居住服务——这在经济性质上与企业购置机器设备相同：都是为了获取未来持续的服务流，而非一次性消耗。因此无论购买者是企业还是家庭，只要是新建住宅，都归入投资。

另外，把新房购买放入I，比放入C更容易追踪和统计——建筑业的产出本来就和企业资本支出一起由统计部门监测。

  • 买新房 → 计入I（有新的生产发生）
  • 买二手房 → 不计入GDP（只是资产转手，没有新生产）
  • 支付租金 → 计入C（购买居住服务，属于消费）

G：政府支出（Government Spending）

各级政府购买商品和服务的支出，不含转移支付。

包括：国防、基础设施建设、公务员薪酬、政府采购等。

不包括：社会保障金、退休金、失业救济等转移支付——这些支付没有对应的新商品或服务被生产。政府支出在美国约占GDP的18%（2014年数据为18.4%，长期均值约20%）。

X-M：净出口（Net Exports）

净出口 = 出口（X）- 进口（M）

出口（Exports）：在本国生产、卖给外国的商品和服务，加入GDP。

进口（Imports）：在外国生产、由本国购买的商品和服务，从GDP中减去——因为消费、投资、政府支出中可能包含了进口商品，必须减去以确保只计算国内生产。

净出口为正 → 贸易顺差（Trade Surplus）：出口 > 进口 净出口为负 → 贸易逆差（Trade Deficit）：进口 > 出口

进口抵消举例：

美国航空公司花$1亿购买空客飞机（欧洲制造）：

  • 投资I +$1亿（美国企业采购）
  • 净出口X-M -$1亿（进口$1亿）
  • GDP净变化 = 0

逻辑：该飞机不是美国生产的，不应计入美国GDP，进出相抵确保了这一点。

支出法从需求侧测量，也可以从生产侧将GDP分为五类：

类别美国占比（2014）说明

1.5 GDP公式总结

支出法：Y = C + I + G + (X - M)
收入法：Y = w + i + r + p
增值法：Y = Σ 各环节增加值

三种方法结果相同——因为每笔市场交易既是买方的支出，也是卖方的收入，也是该环节的增加值。

与宏观框架的衔接

GDP是衡量Dalio框架中"经济总量"最直接的指标。

在Dalio的价格等式中，价格 = 总支出 ÷ 商品总量，GDP支出法的C+I+G+(X-M)正是这个"总支出"的精确分解。信用扩张推高C和I，央行政策影响利率进而影响I，政府财政赤字扩大G——这些都是Dalio框架中债务周期的微观对应。

GDP本身有两个重要局限：① 它是名义值，需要剔除通胀才能得到实际增长（实际GDP vs 名义GDP，后续章节展开）；② 它不衡量财富分配、环境损耗、家庭生产等非市场活动，因此不能直接等同于社会福祉。

2. GDP的局限性与替代指标

2.1 人均GDP（GDP per capita）

在讨论GDP的局限性之前，先引入一个常用的衍生指标。

人均GDP（GDP per capita）：将GDP除以人口数量，用于粗略衡量一个国家的平均生活水平。

GDP per capita = GDP ÷ 人口数量

国际间人口迁移通常从人均GDP低的国家流向高的国家，说明它确实捕捉到了部分生活水平的差异。但它只是一个粗略指标——这正是本节要讨论的核心问题。

名义人均GDP还有一个跨国比较的局限：不同国家物价水平差异悬殊，同样金额在不同国家能买到的东西差异极大。因此跨国比较时，通常需要使用购买力平价调整后的人均GDP（GDP per capita PPP，Purchasing Power Parity）——用统一的"国际价格"重新衡量各国产出。

例： 卢森堡人口极少，名义GDP总量不大，但人均GDP远超美国；按PPP调整后，卡塔尔的排名更高，因为当地物价水平使同样金额能购买更多商品和服务。

2.2 GDP的局限性

GDP是目前最广泛使用的经济衡量指标，但它从设计之初就不是用来衡量社会福祉的——它衡量的是市场产出，而非生活质量（Quality of Life）。两者高度相关，但并不等同。

未被计入的经济活动

非市场交易（Non-market Transactions）：只要没有发生市场交换，GDP就无法捕捉，无论实际价值多高。

  • 自己照顾子女 vs 雇佣保姆：后者计入GDP，前者不计入，但提供的服务完全相同
  • 自己开车送人 vs 打车：后者计入GDP，前者不计入
  • 自己种菜自己吃：不计入GDP

这导致一个有趣的统计现象：随着越来越多女性进入劳动力市场，原本由家庭内部提供的服务（烹饪、育儿等）转移到市场上，GDP随之上升——但这并不意味着社会实际消费的服务总量增加了，只是统计可见性提高了。

地下经济与非法活动：现金交易未申报、黑市交易等均不计入GDP。据估计，美国地下经济规模约占GDP的6.6%（2013年约$2万亿）。

GDP无法衡量的维度

  • 收入不平等（Income Inequality）：人均GDP只是平均值。GDP增长5%可能意味着所有人都增长了5%，也可能意味着少数人大幅增长而多数人几乎没有变化。GDP对分配问题完全失语。
  • 休闲时间：美国人均GDP高于德国，但德国工人每年工作时间比美国工人少数百小时。额外的休假时间不体现在GDP中，但显然是生活质量的一部分。
  • 环境质量（Sustainability）：GDP计入了购买污染控制设备的支出，但不衡量空气和水实际上是否变干净了。一个GDP高速增长但环境严重恶化的国家，GDP数字会呈现繁荣，实际生活质量却可能在下降。
  • 经济弊端（Economic Bads）：任何经济活动造成的、对社会产生负面价值的结果，例如汽车造成的空气污染会损害人类健康和环境；不可持续的经济增长可能会降低一个国家人民的生活质量。
  • 健康与教育的实际水平：GDP计入医疗支出，但不衡量预期寿命是否提高；GDP计入教育支出，但不衡量识字率或学习成果。
  • 科技进步与产品多样性：1950年代无论多有钱，都买不到智能手机。GDP无法衡量技术进步带来的生活质量提升，也无法反映产品种类的增加给消费者带来的福利。

GDP既会高估也会低估生活水平的提升

可能低估：工作时间在过去一个世纪大幅缩短；预期寿命和教育水平显著提升；科技进步带来的价值无法被充分计入。

可能高估：原本由家庭（主要是女性）提供的非市场服务转移到市场后推高了GDP，但实际消费量未必增加；犯罪、交通拥堵、收入不平等加剧等推高了部分GDP，却降低了生活质量。

2.3 相关衍生指标

除GDP外，还有几个相关的经济指标值得了解：

国民生产总值（GNP，Gross National Product）：计算本国公民和企业在全球范围内的产出，无论在哪个国家生产。

GNP = GDP + 本国居民在海外的收入 - 外国居民在本国的收入

GDP关注地理边界，GNP关注国籍。对于有大量人口在海外工作并汇款回国的小国，两者差距可能很大；对美国而言，差距约为0.2%。

国民生产净值（NNP，Net National Product）：GNP减去资本折旧。

NNP = GNP - 折旧（Depreciation）

折旧（Depreciation） 是资本随时间老化而损失的价值。GDP将新资本投资全额计入，但忽略了旧资本的损耗，因此在资本折旧较快的经济体中，GDP可能高估实际经济活动水平。NNP通过扣除折旧，提供了一个更接近"净新增财富"的指标。

国民收入（National Income）：NNP进一步细分，包含所有劳动和资本收入（工资、利润、租金、利息）。

⚠️ 实际使用中不需要记住所有定义，但要注意：在比较不同国家或不同年份的数据时，必须确认比较的是同一个指标（GDP vs GNP vs NNP），否则结论可能失真。

2.4 替代性综合指标

为了弥补GDP的局限，不同机构开发了更全面的生活质量衡量工具：

  • 人类发展指数（HDI，Human Development Index）：联合国开发的综合指标，涵盖健康（预期寿命）、教育（受教育年限）、收入（人均GNI）三个维度。
  • 真实进步指标（GPI，Genuine Progress Indicator）：在GDP基础上，加入志愿服务、家务劳动等正面非市场活动的价值，同时减去犯罪成本、环境破坏、资源耗竭、收入不平等等负面因素。
  • 快乐星球指数（HPI，Happy Planet Index）：综合幸福感调查、预期寿命、社会不平等程度和生态足迹，衡量一个国家在多大程度上实现了可持续的人类福祉。

这些指标与GDP排名往往不一致——GDP排名靠前的国家，在幸福感或可持续发展指标上未必领先。

2.5 GDP的定位

GDP衡量市场产出是准确且有用的。它能有效反映就业状况、收入水平、经济扩张或收缩，是判断短期经济景气程度的最佳单一指标。

但它不是生活质量的精确衡量，在以下场景中需要补充其他维度：比较不同发展阶段的国家、评估增长的可持续性、分析社会公平问题、或判断政策对普通人实际生活的影响。

一个实用的认知框架：GDP是经济活动的温度计，不是幸福的晴雨表。两者通常同向变动，但在边际上会出现背离。

与宏观框架的衔接

Dalio框架里的"经济总量"主要通过GDP来衡量，这在分析短期债务周期（扩张期GDP增长、衰退期GDP收缩）时是合适的工具。但在评估长期生产力增长时，需要注意GDP的局限——技术进步、工作时间缩短、环境改善等对生活质量的实质提升，往往在GDP数字中低估甚至缺失。这也是为什么"人均实际GDP增长"只是衡量长期生产力进步的代理指标，而非直接等同于生活水平的提升。

3.失业率

3.1 人口的分层结构

理解失业率，首先需要理解人口是如何被分层的：

flowchart TD
    A["全体人口"]
    B["适龄人口\n未被监禁、非现役军人"]
    C["劳动力"]
    D["非劳动力\n退休者、在校学生等"]
    E["就业者\n有偿工作（哪怕每周仅1小时）"]
    F["失业者\n无工作 + 过去四周内积极求职"]
    G["丧志工作者\n已放弃求职"]

    A --> B
    B --> C
    B --> D
    C --> E
    C --> F
    F -.->|退出劳动力| G
    G --> D

    H["失业率 UR = 失业人数 ÷ 劳动力总数 × 100%"]
    I["劳动力参与率 LFPR = 劳动力 ÷ 适龄人口 × 100%"]
  • 适龄人口（Eligible Population）：16岁及以上、未被监禁、非现役军人的人口。
  • 劳动力（Labor Force）：适龄人口中，正在工作或积极寻找工作的人。
  • 就业（Employed）：在统计参考周内有偿工作过哪怕一小时，即算就业。
  • 失业（Unemployed）：在参考周内完全没有工作，且在过去四周内积极寻找工作。

⚠️ "积极寻找工作"是关键限定词。若一个人没有工作但在过去四周内没有主动求职（如已放弃寻找），则不计入失业者，而是被归为"非劳动力"。

3.2 核心计算公式

劳动力：

劳动力 = 就业人数 + 失业人数

失业率（Unemployment Rate，UR）：

失业率 = 失业人数 ÷ 劳动力总数 × 100%

劳动力参与率（Labor Force Participation Rate，LFPR）：

LFPR = 劳动力 ÷ 适龄人口 × 100%

LFPR衡量的是适龄人口中实际参与劳动市场的比例，是理解失业率变动背后原因的重要补充指标。

失业率的统计方式（美国）

美国劳工统计局（Bureau of Labor Statistics）每月对约6万个家庭进行调查，询问其就业状态和求职情况，以此推算全国数据。这是抽样调查，不是普查。

3.3 失业率的局限性

失业率最重要的局限在于：它的分母（劳动力）本身会随经济好坏而变化，导致失业率可能误导判断。

经济恶化时，失业率可能被低估

当经济持续低迷，部分失业者因长期找不到工作而放弃求职，成为丧志工作者（Discouraged Workers）。他们退出劳动力统计，导致：

  • 失业人数下降（分子减小）
  • 劳动力总数也下降（分母减小）
  • 失业率可能反而下降——但这并不代表就业状况改善

例： 原有1500万失业者、劳动力1.54亿，失业率9.7%。若500万人因受挫退出劳动力，则失业人数降至1000万，劳动力降至1.49亿，失业率变为6.7%——经济实际上更糟，数字却好看了。

经济好转时，失业率可能被高估

当经济改善，原本退出劳动力的人重新开始求职，重新被计入失业者，导致：

  • 失业人数上升（分子增大）
  • 劳动力总数也上升（分母增大）
  • 失业率可能暂时上升——但这恰恰反映了经济好转，人们重拾求职信心

因此，应同时观察失业率和劳动力参与率，才能更完整地判断就业市场的真实状况。若失业率下降同时LFPR也下降，可能是丧志工作者增加而非就业真正改善。

其他局限

未充分就业（Underemployed）：想要全职工作但只能找到兼职工作的人，被算作就业者，不计入失业率，但他们的资源并未被充分利用。

3.4 三种失业类型

失业率可以分解为两部分：自然失业率和周期性失业。

失业率 = 自然失业率（NRU）+ 周期性失业率

自然失业率（Natural Rate of Unemployment，NRU）

即使经济运行在最优状态、充分就业时，仍然存在的失业率。自然失业率 = 摩擦性失业 + 结构性失业，永远不会等于零。

摩擦性失业（Frictional Unemployment）

求职过程本身存在摩擦和时间成本——投递简历、面试、等待录用——这个过程不是瞬间完成的。即使劳动者的技能完全符合市场需求，也需要时间才能找到合适的岗位。

典型例子：刚毕业的大学生正在寻找第一份工作；从一家公司离职后正在寻找下一份工作的人。

摩擦性失业不是坏事——它意味着劳动者有选择工作的自由。技术进步（如求职网站、招聘App）可以降低摩擦性失业，但不会使其归零。

结构性失业（Structural Unemployment）

劳动者的技能与雇主需求之间存在结构性错配，通常由经济结构的根本性变化引起——新技术取代旧技术、新产业替代旧产业。

典型例子：马车夫在汽车普及后失业；煤矿工人在能源转型后失业；制造业工人被自动化机器替代。

结构性失业往往需要较长时间才能消化，因为劳动者需要学习新技能或迁移到新地区。

周期性失业（Cyclical Unemployment）

由经济周期（衰退与扩张）引起的失业，等于实际失业率与自然失业率之差：

周期性失业率 = 实际失业率 - 自然失业率
  • 衰退期：周期性失业为正，实际失业率高于自然失业率
  • 扩张期：周期性失业可以为负，实际失业率低于自然失业率——此时企业因急需人手而降低招聘标准，雇用了在正常情况下仍在寻职的摩擦性失业者

三种失业类型对比：

类型原因消除方式
摩擦性（Frictional）求职过程需要时间改善信息匹配效率
结构性（Structural）技能与需求错配再培训、产业转型
周期性（Cyclical）经济衰退财政/货币政策刺激需求

3.5 充分就业与自然失业率

充分就业（Full Employment） 不等于零失业，而是经济在PPC上高效运行、失业率等于自然失业率的状态。此时只有摩擦性和结构性失业存在，没有周期性失业。

  • 若实际失业率 > NRU：经济运行在PPC内侧，存在资源闲置
  • 若实际失业率 < NRU：经济短暂运行在PPC外侧，不可持续

NRU本身也会随时间缓慢变化——若结构性变化加速（如技术革命），NRU可能上升；若求职匹配效率提升，NRU可能下降。

与宏观框架的衔接

在Dalio框架中，失业率是短期债务周期最直接的社会表现之一。信用收缩→企业收入下降→裁员→周期性失业上升；央行降息或政府财政刺激→需求回升→周期性失业下降。摩擦性和结构性失业则对应Dalio框架中"生产力增长"的结构性约束——技术替代和产业转型会持续产生结构性失业，这是经济进步不可避免的副产品，无法通过货币或财政政策消除，只能通过教育和再培训来缓解。

4. 通货膨胀

通货膨胀（Inflation）：经济中整体价格水平的持续上涨，导致货币购买力下降。

注意两个限定词：整体（不是某个商品涨价）和持续（不是短期波动）。

通胀率（Inflation Rate）：价格水平从一个时期到下一个时期的百分比变化。

通胀率 = (新价格水平 - 旧价格水平) ÷ 旧价格水平 × 100%

⚠️ 历史上"通货膨胀"这个词原本指的是货币供应的膨胀（货币型），现代经济学中通胀默认指价格通胀。两者相关但不完全等同——货币供应增长不一定立刻导致价格上涨，短期内还可能受供给冲击等因素影响。

4.1 CPI：通胀的测量方法

篮子商品的逻辑

测量通胀需要一个基准——不能只看某一种商品的价格，而要看代表性消费者购买的一整套商品和服务的价格变化。

消费者价格指数（Consumer Price Index，CPI）：追踪一个固定"篮子"的商品和服务的总成本随时间的变化，衡量典型城市家庭的生活成本变化。

市场篮子（Market Basket）：用于计算CPI的固定商品组合，每年保持不变。篮子中各类商品的权重，基于实际消费者的支出比例——消费者花更多钱的商品，对CPI的影响也更大。

基准年（Base Year）：作为比较参照的年份，其CPI被设定为100。选择哪一年作为基准年是任意的，不影响通胀率的计算结果。

CPI的计算步骤

  1. 确定篮子商品的组成和数量
  2. 计算基准年购买该篮子的总成本
  3. 计算目标年购买该篮子的总成本
  4. CPI = (目标年成本 ÷ 基准年成本) × 100

例（简化版）：

假设篮子只包含两种商品：

商品数量基准年单价目标年单价
面包10条$2$3
牛奶5升$4$4
基准年成本 = 10×$2 + 5×$4 = $40
目标年成本 = 10×$3 + 5×$4 = $50

CPI（目标年）= ($50 ÷ $40) × 100 = 125

CPI为125意味着：相比基准年，购买同样的篮子现在需要多花25%。

通胀率的计算

有了两个时期的CPI，通胀率的计算就是标准的百分比变化：

通胀率 = (CPI₂ - CPI₁) ÷ CPI₁ × 100%

例： 若某年CPI为125，下一年CPI为130：

通胀率 = (130 - 125) ÷ 125 × 100% = 4%

⚠️ 不能直接用CPI数字相减得到通胀率（如130-125=5%），必须除以基准期CPI。两者接近时误差不大，但精确计算必须用公式。

⚠️ 通胀复利计算：若每年通胀2%，10年后价格不是上涨20%，而是约22%（复利效应）。

4.2 名义变量与实际变量

这是理解通胀影响的核心工具，也是连接通胀与投资决策的关键。

名义变量（Nominal Variables）：未经通胀调整的账面数字，如名义工资、名义利率。

实际变量（Real Variables）：经通胀调整后，反映真实购买力的变量。

实际利率 ≈ 名义利率 - 通胀率
实际工资增长 ≈ 名义工资增长 - 通胀率

例： 银行存款名义利率3%，通胀率5%，实际利率 = 3% - 5% = -2%。账面上在赚钱，实际购买力在缩水。

购买力（Purchasing Power）：一定数量的货币实际能买到的商品量。通胀率越高，货币购买力下降越快。

4.3 CPI的局限性

CPI是目前最广泛使用的通胀衡量工具，但它有两个系统性偏差，导致它倾向于高估真实生活成本的上升。

替代偏差（Substitution Bias）

CPI假设消费者每年购买完全相同的篮子商品，但现实中，当某种商品价格上涨时，消费者会转向相对便宜的替代品。固定篮子无法捕捉这种行为调整，因此高估了生活成本的上升。

例： 牛肉涨价，消费者转向鸡肉。CPI仍按原来的牛肉购买量计算，但实际消费者已经降低了牛肉支出。

质量/新商品偏差（Quality/New Goods Bias）

固定篮子无法充分反映商品质量的提升和新商品的出现。

质量提升：今天的汽车比20年前的汽车有更多安全配置，价格更高但功能也更强。CPI可能将价格上涨全部计为通胀，而忽略了质量改善的部分。

新商品：篮子商品的更新滞后于市场。新产品刚上市时价格最高，此后逐渐下降。但CPI篮子的更新滞后，等新产品被纳入时，价格已经下降了一段时间，导致CPI错过了这段价格下降过程，从而高估了通胀。

综合两种偏差，CPI每年可能高估真实通胀约0.5个百分点。

4.4 根据目的选择不同价格指数

这一节引入多种价格指数，核心逻辑是：没有"最好"的指数，只有"最适合这个问题"的指数。

指数英文衡量对象主要用途
消费者价格指数CPI城市家庭购买的篮子商品衡量家庭生活成本，新闻报道的"通胀率"
核心通胀指数Core CPICPI剔除食品和能源后的部分央行政策决策，过滤短期波动
GDP平减指数GDP Deflator所有GDP组成部分的价格最全面的价格水平衡量，用于计算实际GDP
生产者价格指数PPI生产者购买的原材料和中间品CPI的先行指标，预判消费端通胀走向
就业成本指数ECI劳动力市场的工资衡量劳动力成本通胀

为什么需要核心通胀：

食品和能源价格因天气、地缘政治等短期因素波动剧烈。若一场飓风推高了一个月的油价，这不代表底层通胀趋势改变，央行不应据此调整利率政策。核心通胀过滤掉这些噪音，让政策制定者看到更持久的通胀趋势。

为什么GDP平减指数不适合衡量生活成本：

它包含了家庭不购买的商品（如军用飞机、工厂设备），因此不能准确反映普通消费者的生活成本变化。但它是计算实际GDP（剔除通胀后的GDP）最准确的工具。

PPI领先于CPI的逻辑：

生产成本上升（PPI上涨）通常在数月后才会传导到消费端价格（CPI上涨）。观察PPI走势，可以提前预判CPI的变动方向。

4.5 通胀的赢家与输家

预期通胀与非预期通胀

通胀对不同群体的影响，取决于一个关键前提：通胀是否在预期之内。

预期通胀（Anticipated Inflation）：借贷双方签订合同时已将通胀预期纳入名义利率，实际影响有限。若双方都预期通胀5%，名义利率就会包含这5%的补偿，实际利率大体不变，财富不会被意外再分配。

非预期通胀（Unanticipated Inflation）：实际通胀偏离预期，导致财富在借贷双方之间发生非预期的再分配。这才是通胀真正产生"赢家与输家"的根本原因。

借贷双方的影响机制

以固定利率借贷为例：今天贷款人借出$100，约定一年后收回$110（名义利率10%）。此时$100恰好等于一篮子商品的价格。

通胀情景一年后篮子价格$110能买多少篮子贷款人实际回报
无通胀$1001.1篮子+10%
温和通胀（2%）$102约1.08篮子约+8%
极端通胀（120%）$2200.5篮子-50%
极端通缩（45%）$552篮子+100%

规律：

  • 非预期通胀 → 借款人受益、贷款人受损：借款人用购买力更低的货币还债，实际债务负担下降；贷款人收回的钱购买力低于借出时。
  • 非预期通缩/通胀放缓 → 贷款人受益、借款人受损：贷款人收回的钱购买力高于预期；借款人须用购买力更高的货币还债，实际利率负担上升。

其他受影响群体

  • 受损方：
    • 持有大量现金者：现金购买力随通胀直接缩水
    • 固定名义收入者（如固定养老金）：实际购买力被持续侵蚀，通胀1-2%看似温和，数十年复利累积后影响可观
  • 受益方：
    • 持有实物资产者（房产、黄金）：资产价格通常随通胀上升，实际财富相对保值

对投资的含义： 当通胀上升而名义利率不变时，实际利率下降。持有不产生现金流的黄金机会成本降低，吸引力上升——这正是黄金与实际利率负相关的机制来源。

4.6 通胀的危害与极端情形

通胀的危害

  • 价格信号模糊
    价格是市场经济中传递供需信息的信号。高通胀时，价格变动中有多少是通胀造成的、有多少是真实供需变化，变得难以分辨。企业和消费者无法准确读取价格信号，资源配置效率下降，市场向均衡调整的速度也会减慢。
  • 长期规划困难
    通胀增加了未来的不确定性。退休规划、企业投资决策、长期借贷合同，都需要对未来价格水平做出预判。通胀越高、越不稳定，长期规划的难度和风险就越大。企业可能将精力转向对冲通胀风险，而非提升生产效率。

通胀放缓（Disinflation）

通胀率在下降但仍为正值，价格仍在上涨，只是涨得慢了。政策制定者说"降低通胀"时，目标是通胀放缓，而非通缩。

通缩

价格水平绝对下降，即通胀率为负。

通缩表面上看价格下跌对消费者有利，但往往伴随严重的经济问题：

消费者预期"明天更便宜"→ 推迟购买 → 企业收入下降 → 裁员 → 更少人消费 → 价格继续下跌 → 螺旋强化

通缩还使货币政策失效：名义利率无法大幅为负，而通缩意味着实际利率自动上升，抑制借贷和投资。这也是大多数政策制定者将避免通缩视为比控制通胀更优先目标的原因。

恶性通胀（Hyperinflation）

恶性通胀是极端的高通胀，通常与政府过度印钞有关，往往发生在计划经济向市场经济转型期间或政府财政完全失控时。在恶性通胀下，货币几乎失去价值，价格信号完全崩溃，经济活动瘫痪，社会稳定受到严重威胁。

德国1923年和津巴布韦2000年代是教科书案例。

滞胀（Stagflation）

滞胀是通胀与经济停滞同时出现的异常状态——价格上涨的同时，经济增长放缓、失业率上升。

经典触发机制是供给侧冲击（如石油禁运）：

供给骤减 → 价格上涨（通胀）→ 生产成本上升 → 企业利润下降 → 裁员 → 需求下降 → 经济停滞

滞胀是央行最难应对的情形：加息可以压制通胀，但会进一步打压已经疲软的经济；降息可以刺激经济，但会加剧通胀。两个目标相互冲突，货币政策没有两全其美的解法。

1970年代美国石油危机是最典型的历史案例。

与宏观框架的衔接

在Dalio框架中，通胀是信用扩张的直接产物——更多的信用创造意味着更多的总支出，推动价格水平上升。CPI是衡量这一过程最直接的指标。

实际利率（名义利率 - 通胀率）是理解黄金、债券、汇率等资产价格的核心工具——这也是本节"名义vs实际变量"的实际意义。持续监控通胀数据（CPI、核心通胀、PPI）并理解各指标的含义，是判断央行政策走向、进而判断资产配置时机的基础能力。

5. 名义GDP与实际GDP

5.1 为什么需要区分名义与实际

GDP用货币来计量产出，但货币的购买力会随通胀变化。如果今年GDP从$1,000增长到$1,200，增幅20%，这可能是因为产出真的增加了，也可能只是价格上涨了，或两者兼有。

以一个只生产苹果的简化经济体为例：

年份一年份二
GDP$1,000$1,200
苹果单价$0.50/磅$0.55/磅
苹果产量2,000磅2,182磅

名义GDP增长了20%，但实际产量只增长了约9%。若不区分名义与实际，就会高估经济真实增长。

名义GDP与实际GDP的关系

在基准年，名义GDP = 实际GDP（平减指数 = 100，两者完全相同）。

基准年之后，若有通胀，名义GDP会随价格上升而增大，与实际GDP的差距逐渐拉开。名义GDP曲线的斜率看起来更陡，但这种"增长"有相当部分只是价格上涨的幻觉。

⚠️ 看到GDP增长数字时，必须先确认是名义还是实际。名义GDP增长不一定代表产出真正增加——可能只是通胀推高了价格。只有实际GDP增长才代表真实产出的扩张。

5.2 定义

名义GDP（Nominal GDP）

用当年实际价格计算的GDP，也称"现价GDP"。它反映当年的市场交易总额，但包含了价格变动的影响，不能单独反映产出数量的变化。

实际GDP（Real GDP）

用某一基准年的价格计算的GDP，也称"不变价GDP"。它剔除了价格变动的影响，真实反映产出数量的变化，是衡量经济真实增长的指标。

GDP平减指数（GDP Deflator）

追踪一国境内所有最终商品和服务平均价格水平的价格指数。基准年的平减指数定义为100，其他年份的平减指数反映相对于基准年的价格变化程度。

GDP平减指数由名义GDP和实际GDP的比值推导而来：

GDP平减指数 = (名义GDP ÷ 实际GDP) × 100
  • 平减指数 > 100：价格高于基准年，存在通胀
  • 平减指数 < 100：价格低于基准年，存在通缩

5.3 实际GDP的计算方法：基准年价格加权法

将目标年的各商品产量乘以基准年的价格，加总得到实际GDP。这样可以剔除价格变动，只反映产量的真实变化。

计算步骤：

  1. 确定基准年，记录基准年各商品价格
  2. 记录目标年各商品产量
  3. 用基准年价格 × 目标年产量，加总

沿用上面的苹果例子：年份二产量2,182磅，用年份一价格$0.50计算：

实际GDP（年份二，以年份一价格）= 2,182 × $0.50 = $1,091

与名义GDP（$1,200）相比，实际增长只有$91，即约9%，而非表面上的20%。

局限： 这种方法可能高估实际GDP增长。若某商品价格随时间持续下降（如电子产品），用很久以前的高价作为基准，会夸大该商品在GDP中的贡献，使实际GDP增长看起来比实际更高。

例：

假设苹果价格从年份一的$0.50降到年份三的$0.25，但产量从2,000磅增加到2,200磅。

用年份三当年价格计算：2,200 × $0.25 = $550 用年份一基准价格计算：2,200 × $0.50 = $1,100

同样是增产200磅，用基准年价格算出来的产值是$1,100，比用当年价格算出来的$550高出一倍。实际GDP增长被明显高估了。

⚠️ 注意：这里高估的是产出的货币价值，不是产量本身。产量是客观的，但用什么价格来衡量它的价值，会产生不同的结果。实际操作中，统计机构会通过定期更新基准年或使用更复杂的链式加权方法来减少这种偏差。

与宏观框架的衔接

Dalio框架里反复强调名义值与实际值的区分——工资名义上在涨，但实际购买力可能在缩水；债务以名义值计，但真实偿债压力由实际值决定。名义GDP和实际GDP的区分是这一原则在宏观统计层面的具体体现。实际GDP增长是衡量一国生产力真正提升的核心指标。

6. 商业周期

将实际GDP随时间的变化画出来，可以观察到两个叠加的趋势：

长期趋势线：由人口增长和生产力提升共同驱动，实际GDP在长期内稳定上升。

短期波动：实际GDP围绕长期趋势线上下波动，有时高于趋势线，有时低于趋势线。

这种围绕趋势线的短期波动，就是商业周期（Business Cycle）。

"周期"这个词有一定误导性——它并不是像正弦波那样规律、可预测的模式。商业周期的幅度、持续时间都高度不规则，很难预测。相邻两次顶峰或谷底之间的时间间隔并不固定，这正是商业周期难以预测的根本原因。

6.1 商业周期的四个阶段与两个转折点

阶段/转折点定义
扩张（Expansion）实际GDP增加，失业率下降
顶峰（Peak）扩张结束、收缩开始的转折点
衰退（Recession）实际GDP下降，失业率上升
谷底（Trough）衰退结束、复苏开始的转折点
复苏（Recovery）从谷底开始，GDP重新上升，直至回到潜在产出水平

衰退与萧条（Depression）：衰退指实际GDP显著下滑；若衰退程度极深、持续时间极长，则称为萧条。1930年代的大萧条是最典型的案例。

在美国，商业周期的官方记录由私立研究机构 全国经济研究局（NBER） 负责。值得注意的是，严重衰退的实际影响往往在NBER宣布的官方结束日期之后仍持续相当长时间。

6.2 潜在产出与产出缺口

潜在产出（Potential Output）：经济体在所有资源被充分有效利用时能够达到的实际GDP水平，也称"充分就业产出"。此时失业率恰好等于自然失业率（NRU），没有周期性失业。在商业周期图中，潜在产出对应长期增长趋势线。

产出缺口（Output Gap）：实际产出与潜在产出之间的差值。

  • 正产出缺口（Positive Output Gap）：实际产出 > 潜在产出，商业周期曲线在趋势线上方。此时经济"过热"——失业率低于NRU，企业超负荷运转，资源被过度使用。这种状态不可持续，最终将回落至顶峰后进入衰退。
  • 负产出缺口（Negative Output Gap）：实际产出 < 潜在产出，商业周期曲线在趋势线下方。此时失业率高于NRU，经济运行在PPC内侧，存在资源闲置。

6.3 与PPC的对应关系

商业周期的各阶段可以用PPC直观表示：

商业周期位置PPC对应位置含义
实际产出 = 潜在产出PPC上的点充分就业，资源有效利用
正产出缺口（顶峰附近）PPC外侧的点资源过度使用，不可持续
负产出缺口（衰退期）PPC内侧的点资源闲置，存在周期性失业
复苏从PPC内侧移向PPC上的点产出恢复，失业率回落

6.4 扩张≠经济增长

这是最常见的误区之一。

扩张是从衰退谷底回升的过程，是短期波动中实际产出的增加，对应商业周期曲线的上行段。即使经济处于扩张阶段，如果实际产出尚未回到潜在产出水平，仍然存在负产出缺口。

经济增长（Economic Growth） 是潜在产出本身的提升——趋势线上移，或PPC整体外移。这来自人口增长、技术进步、资本积累等长期结构性因素，与短期的周期性波动无关。

例： 若一国从$2万亿的潜在产出水平衰退至$1.9万亿，随后复苏回到$2万亿，这只是短期波动的恢复，不是经济增长。若潜在产出本身从$2万亿提升至$2.5万亿，才是真正的经济增长。

6.5 人类情绪与商业周期

传统经济学模型将人类情绪排除在外，但情绪可能是驱动商业周期最重要的因素之一。

扩张阶段的情绪演变：

衰退刚结束时，人们仍心有余悸，对复苏持怀疑态度，不敢消费和投资。随着时间推移，痛苦的记忆逐渐淡化，人们开始重建信心，消费和投资增加，经济进一步扩张。到了扩张后期，距离上次衰退已经很远，人们开始低估风险，过度乐观，倾向于将近期的增长外推至未来。

顶峰与衰退阶段的情绪演变：

乐观 → 兴奋 → 狂热 → 欣喜若狂 → 焦虑 → 否认 → 恐惧 → 恐慌 → 绝望

谷底与复苏阶段的情绪演变：

绝望 → 沮丧 → 希望 → 释然 → 乐观（新一轮循环）

这个情绪周期与股市周期高度吻合，也与实体经济周期紧密相连。从投资角度看，"最好的投资时机"往往出现在情绪最悲观的谷底附近，而"最危险的时机"往往出现在情绪最狂热的顶峰附近。

⚠️ 传统经济学的总供给和总需求模型不包含情绪因素，这是这些模型的重要局限之一。行为经济学（Behavioral Economics）是专门研究情绪和非理性行为对经济影响的新兴领域。

与宏观框架的衔接

商业周期是Dalio框架中短期债务周期（5-8年）的统计表现。信用扩张推动产出高于趋势线（正产出缺口），信用收缩导致产出低于趋势线（负产出缺口）。本节的情绪分析与Dalio框架高度吻合——Dalio同样强调市场参与者的情绪和预期是推动周期的核心力量，而非单纯的经济变量。潜在产出的长期上升趋势，则对应Dalio框架中"生产力增长"这条底层趋势线。
